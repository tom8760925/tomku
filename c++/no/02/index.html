C++語言的練習-型態

2024-03-28

型態

這篇是C++的練習筆記第02篇
記錄C++有哪些型態
在型態前面加上unsigned，會變強制沒有負號
在型態前面加上signed，會變強制有負號

種類名稱記憶體空間加unsigned加signed
數值int4 Bytes可可
short2 Bytes可可
long8 Bytes可可
long long8 Bytes不可不可
浮點數float4 Bytes不可不可
double8 Bytes不可不可
long double16 Bytes不可不可
字元char1 Bytes可可
無種類void不可不可
布林bool1 Bytes不可不可
Unicodewchar_t不可不可

c++有string，可以使用但需要宣告，才能使用string。

#include <string>
using namespace std;

使用方式

#include <iostream> 
#include <string>
using namespace std;
int main(){
    string d="AAA"; 
    cout << "p：" << d << endl;
    return 0;
}

結果

p：AAA
