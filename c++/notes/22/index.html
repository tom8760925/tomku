vscode執行多個檔案

這篇是C++的筆記第22篇
在使用vscode寫c++時發現，預設是不能執行多個檔案，但改了tasks.json檔案裡的設定就可以執行了

將"${file}"改成"${fileDirname}/*.cpp"

如果事使用Code Runner進行編譯的，可以在settings.json跟改設定就可以執行了

將"cpp": "cd $dir && g++ $fileName -o $fileNameWithoutExt -pthread && $dir$fileNameWithoutExt" 
改成
"cpp": "cd $dir && g++ $dir*.cpp -o $fileNameWithoutExt -std=c++17 -pthread && $dir$fileNameWithoutExt",

這些方法只能執行同一個資料夾內的檔案，其他資料夾是不能的。
