C++語言的筆記-pair

2025-02-27

pair

這篇是C++的筆記第82篇

pair是一種容器，能存兩個不同類型，但又跟map不一樣只能存一組。

#include <iostream>
#include <utility>

int main()
{
    // 宣告
    std::pair<int, std::string> p1(1, "a");
    std::pair<int, std::string> p2 = std::make_pair(1, "a");
    std::cout << "===1===" << std::endl;
    // 使用
    std::cout << p1.first << std::endl;
    std::cout << p1.second << std::endl;

    return 0;
}

輸出

===1===
1
a
