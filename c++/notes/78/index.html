C++語言的筆記-std::function

2025-02-26

std::function

這篇是C++的筆記第78篇

std::function是將func轉換成變數。

#include <iostream>
#include <functional>

void func(int n)
{
    std::cout << n << std::endl;
}

int main()
{
    std::cout << "====1===" << std::endl;
    std::function<void(int)> _func1 = func;
    _func1(5);
    std::cout << "====2===" << std::endl;
    std::function<void(int)> _func2 = [](int n)
    { std::cout << n << std::endl; };
    _func2(10);

    return 0;
}

輸出

====1===
5
====2===
10
