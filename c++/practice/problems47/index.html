C++語言的練習-練習實作Maximum Product Subarray

2025-07-31

Maximum Product Subarray

這篇是C++的練習實作第problems47篇

練習leetcode的Maximum Product Subarray

過程

因為這題是要計算子陣列的乘機，而且不能跳著乘。
例如2,3,0,4，那答案會是6不會是12，因為3和4之間有0，不能跳著乘。
而且有可能會出現負負得正，所以這時需要兩的值代表現在相乘最大值和最小值。
而且因為可能會是負負得正，所以在值小於0時會進行交換，讓可能是負數的值與負數的值相乘。 執行過程：

1

n=1
nums[1]=3
maxr=max(3,2*3)
minr=min(3,2*3)
r=max(2,6)

2

n=2
nums[2]=0
swap(maxr,minr)
maxr=max(0,3*0)
minr=min(0,6*0)
r=max(6,0)

3

n=3
nums[3]=4
maxr=max(4,0)
minr=min(4,0)
r=max(6,4)

最後得出6

結果

#include <iostream>
#include <vector>
#include <string>
#include <algorithm>

class Solution
{
public:
    int maxProduct(std::vector<int> &nums)
    {
        int r = nums[0];
        int maxr = nums[0];
        int minr = nums[0];
        for (int n = 1; n < nums.size(); n++)
        {
            if (nums[n] < 0)
            {
                std::swap(maxr, minr);
            }
            maxr = std::max(nums[n], maxr * nums[n]);
            minr = std::min(nums[n], minr * nums[n]);
            r = std::max(r, maxr);
        }
        return r;
    }
};

int main()
{
    Solution s;
    std::vector<int> nums = {2, 3, -2, 4};
    std::cout << s.maxProduct(nums) << std::endl;
    return 0;
}
