其他的筆記-x11vnc

2024-09-02

x11vnc

這篇是x11vnc的筆記 安裝x11vnc

sudo apt-get install x11vnc

設定密碼

x11vnc -storepasswd

啟動x11vnc，會顯示port號

x11vnc -forever -usepw -localhost -rfbport 5901 -display :0
-usepw 使用密碼
-forever 不會斷開時關閉
-localhost 限本地端連線
-rfbport 5901 連線的port號

ssh連線

建立通道

ssh -fNL 5961:localhost:5901(設定port號:localhost:vnc port號) user@ip

進行vnc連線，ip要輸入localhost，port要輸入5961
