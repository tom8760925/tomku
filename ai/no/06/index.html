AI的函數

這篇是AI的練習筆記第06篇

在訓練AI有各式各樣函數輔助訓練，有過程或結果或驗證

啟動函數

啟動函數主要使用在類神經網路中，讓類神經網路可以判別非線性關係，但機器學習也能會使用。 種類為：Step、Sigmoid、Tanh、ReLU、Softmax

步階函數(step function)

主要功能為輸入大於0時輸出1，小於等於0輸出0。
在類神經網路中是當作感知機（Perceptron）使用

Sigmoid

主要功能是將計算結果轉換成機率。
如果輸入2.5
1/(1+2.71828^-2.5) = 0.92414

Tanh

主要功能是將計算結果轉換成1到-1之間。
如果輸入2.5
(2/2+(2.71828^(-2*2.5)))-1=0.98656

ReLU

主要功能為輸入大於0時輸出輸入值，小於等於0輸出0。 在類神經網路中是當作感知機（Perceptron）使用

Softmax

主要功能是將多個輸出轉換機率，最後相加會等於1

如果輸入2.0,1.0,0.1

計算指數
2.71828^2.0=7.389
2.71828^1.0=2.718
2.71828^0.1=1.105

指數總和
7.389+2.718+1.105=11.212

計算機率
7.389/11.212=0.659
2.718/11.212=0.242
1.105/11.212=0.099

最終加總會等於1
0.659+0.242+0.099=1.0

損失函數或目標函數

用來判斷預測和實際的差距越小越好。 種類：mse、mae、Huber Loss、cross-entropy、Cosine Similarity、Kullback-Leibler Divergence

mse

計算平均誤差。 如果有三個結果

實際 預測
300	280
500	510
400	450

計算誤差平方

(280-300)^2 = -20^2 = 400
(510-500)^2 = 10^2 = 100
(450-400)^2 = 50^2 = 2500

計算平均值
(400+100+2500)/3=1000
mse = 1000

mae

用來衡量迴歸模型表現的損失函數。

實際 預測
300	280
500	510
400	450

計算誤差絕對值

|280-300| = 20 = 20
|510-500| = 10 = 10
|450-400| = 50 = 50

計算平均值
(20+10+50)/3=26.67 mae = 26.67

Huber Loss

根據誤差的大小來決定如何計算損失，需要設定閾值決定要使用mse、mae

實際 預測
300	280
500	510
400	450

閾值10

計算誤差

280-300 = -20
510-500 = 10
450-400 = 50

絕對值後小於等於10使用mse和大於10使用mae

|-20|=20
|10|^2=100
|50|=50

計算平均值
(20+100+50)/3=56.67 Huber Loss = 56.67

cross-entropy

用來衡量模型預測的機率分佈與真實機率分佈之間的差異。 例子：

真實標籤：貓=0, 狗=1, 鳥=0
預測機率：0.8, 狗=0.15, 鳥=0.05
−[(0×log(0.8))+(1×log(0.15))+(0×log(0.05))]
−[0+log(0.15)+0]=−(−1.897)≈1.897

輸出為1.897

Cosine Similarity

用來衡量兩個向量有多麼相似的指標。

例子：

A = [2, 5, 1]
B = [3, 6, 2]

# 計算點幾
(2 * 3) + (5 * 6) + (1 * 2) = 38

# 計算長度
A = 根號(2^2+5^2+1^2) = 5.477
B = 根號(3^2+6^2+2^2) = 7

# 計算餘弦相似度
38/(5.477*7) = 0.991

Kullback-Leibler Divergence

衡量兩個機率分佈之間差異

例子：

真實數據p=[0.5,0.3,0.2]
預測數據q=[0.7,0.2,0.1]
0.5*log(0.5/0.7)=0.5×log(0.714)=0.5×(−0.336)=−0.168
0.3*log(0.3/0.3)=0.3×log(1.5)=0.3×(0.405)=0.122
0.2*log(0.2/0.1)=0.2×log(2)=0.2×(0.693)=0.139
# 計算總 KL 散度
−0.168+0.122+0.139≈0.093

優化器

訓練模型的核心引擎，設定權重和找出最小損失函數
種類：SGD、Adagrad、RMSprop、Adadelta、Adam、AdamW、Nadam

SGD

固定學習率。

Adagrad

自動調整學習率，為每個參數分配不同的學習率。

RMSprop

使用最近的梯度來調整學習率。

Adadelta

使用一個移動視窗來計算梯度的平方均值。

Adam

同時考慮了梯度的一階動量和二階動量。

AdamW

修復了Adam在權重衰減處理上的錯誤。

Nadam

將Adam的動量部分換成了Nesterov動量。

度量指標

用來評估模型在測試集或驗證集上的最終表現。
種類：Accuracy, Precision, Recall, F1-Score, ROC AUC, MSE, MAE。

Accuracy

有多少比例是正確的

95(正確預測)/100(總樣本數) = 0.95。

Precision

有多大的信心這個預測是正確的

5(正確預測)/(5(正確預測)+3(錯誤預測))(全部預測)=0.625

Recall

模型成功找到所有真實正樣本的能力

5(正確預測)/(5(正確預測)+5(漏掉預測))(全部預測)=0.5

F1-Score

用來衡量分類模型表現的指標

2*((0.625(Precision)*0.5(Recall))/0.625(Precision)+0.5(Recall))=0.556

ROC AUC

評估二元分類器

MSE

計算平均誤差。

MAE

用來衡量迴歸模型表現的損失函數。
