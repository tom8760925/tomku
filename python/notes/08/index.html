matplotlib

這篇是python的筆記第08篇

安裝

pip install matplotlib

測試

import matplotlib.pyplot as plt

fig, ax = plt.subplots()
ax.plot([0,1],[1,0])
plt.show()

圖

圖表類型

線圖 (Line Plot)

用來展示數據的趨勢或變化

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.plot(x_values, y_values)
plt.show()

圖

散佈圖 (Scatter Plot)

顯示數據點在二維平面上的分佈，以及兩個變量（X、Y）之間的關係。

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.scatter(x_values, y_values)
plt.show()

圖

階梯圖 (Step Plot)

使用垂直和水平的線段來表示數據值在特定點發生跳躍式變化的情況。

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.step(x_values, y_values)
plt.show()

圖

對數-對數圖 (Log-Log Plot)

用來表示倍數差距，例如1,10,100,1000，用在例如分貝

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.loglog(x_values, y_values)
plt.show()

圖

填充區域 (Filled Area)

填充兩條曲線之間的區域

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.fill_between(x_values, y_values)
plt.show()

圖

長條圖 (Bar Chart)

最適合比較不同類別之間的量

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.bar(x_values, y_values)
plt.show()

圖

莖葉圖 (Stem Plot)

展示離散序列數據或數字信號

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.stem(x_values, y_values)
plt.show()

圖

事件圖 (Event Plot)

視覺化事件的發生時間或位置

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.eventplot(x_values)
plt.show()

圖

圓餅圖 (Pie Chart)

展示部分與整體的比例關係

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.pie(x_values)
plt.show()

圖

垂直線條 (Vertical Lines)

標示特定 X 座標的事件或範圍

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.plot(x_values,y_values)
plt.vlines(x=[2, 4],ymin=0,ymax=25)
plt.show()

圖

填充多邊形 (Filled Polygon)

填充由閉合路徑定義的任意區域

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.fill(x_values,y_values)
plt.show()

圖

六邊形分箱圖 (Hexagonal Binning Plot)

視覺化二維數據的密度

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.hexbin(x_values,y_values)
plt.show()

圖

直方圖 (Histogram)

數據分佈的形狀

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.hist(x_values)
plt.show()

圖

階梯圖 (Stairs Plot)

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

plt.stairs(x_values)
plt.show()

圖

二維數據陣列

二維的數值資料轉換成一張彩色圖片

import matplotlib.pyplot as plt
import numpy as np

x_values = np.array([1, 2, 3, 4, 5])
y_values = np.array([5, 15,10 , 20, 0])

values=[x_values,y_values]

plt.imshow(values)
plt.show()
