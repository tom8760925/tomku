tensorflow-圖像AI

這篇是python的筆記第15篇

圖像AI層的內容

模型層數種類

特徵擷取：

  1. 卷積層
  2. 池化層

特徵分類：

  1. 攤平層
  2. 全連接層

卷積層

找出圖片的特徵

使用

input_shape是指將輸入圖片限制到32x32並rgb色 32的意思是用32種不同權重找出特徵 (3, 3)是以3x3大小找出特徵 activation=‘relu’ 將小於0轉換成0 strides=(4, 4) 位移量 padding=‘same’ 周圍補上零像素

layers.Conv2D(32, (3, 3), activation='relu', input_shape=(32, 32, 3))
layers.Conv2D(96, (11, 11),strides=(4, 4), activation='relu', input_shape=(224, 224, 3))
layers.Conv2D(256, (5, 5), padding='same',  activation='relu')

過程

layers.Conv2D(2, (3, 3), activation='relu', input_shape=(3, 3, 3))

以3x3大小圖舉例

[[20,30,1],
[10,3,15],
[25,5,8]]

垂直線條

[[1,0,-1],
[1,0,-1],
[1,0,-1]]

水平線條

[[1,1,1],
[0,0,0],
[-1,-1,-1]]

在相乘後輸出

[[[31],
  [13]]]

31是乘以垂直線條後 13是乘以水平線條後

輔助層

BatchNormalization

將資料逕行整理以便後續分析

layers.BatchNormalization()

Dropout

0.5的機率隨機關閉輸出

layers.Dropout(0.5),

池化層

減少特增數量

使用

pool_size=(3, 3) 將數值以3x3大小找出最大值 strides=(2, 2) (2,2)大小位移

layers.MaxPooling2D(pool_size=(3, 3), strides=(2, 2)),

過程

在卷積層取得資料

[[30,30,1,2],
[10,3,15,24],
[25,5,8,10],
[10,3,21,24]]

以2x2大小找尋最大值輸出

[[30,24],
[25,24]]

攤平層

將二維轉換一維

layers.Flatten()

全連接層

最後輸出

將攤平層整理出64的輸出

layers.Dense(64, activation='relu')

最後整理輸出訓練集所分類的10個

layers.Dense(10, activation='softmax')

最後輸出

會將輸入值進行判斷，輸出訓練集所分類的機率。

編譯

優化器

種類： Adadelta Adafactor Adagrad Adam 優先使用 AdamW Adamax Ftrl Lion LossScaleOptimizer Nadam Optimizer RMSprop SGD

optimizer='adam'

損失函數

種類： BinaryCrossentropy 二元分類 SparseCategoricalCrossentropy 多類別分類 單一值輸出 CategoricalCrossentropy 多類別分類 多數值輸出 MeanSquaredError 回歸 注重異常值 MeanAbsoluteError 回歸 不注重異常值

loss=tf.keras.losses.SparseCategoricalCrossentropy(from_logits=True)

評估指標

分類

以狗和貓舉例
有1000隻動物其中狗50隻和貓950隻
判別100隻其中判別出40狗

種類： Accuracy 40+950/1000 Precision 40/50 Recall 40/100 AUC 會給數值0~1之間大於0.5代表是可以的越接近1代越好

回歸

種類： MeanSquaredError 每個誤差值平方後再求平均 MeanAbsoluteError 每個誤差值的絕對值後再求平均

metrics=['accuracy']

訓練

train_images, train_labels 訓練集 epochs 履歷次數 validation_data=(test_images, test_labels) 測試資料

history = model.fit(train_images, train_labels, epochs=1,validation_data=(test_images, test_labels))

評估

評估訓練的模型

test_loss, test_acc = model.evaluate(test_images,  test_labels, verbose=2)
