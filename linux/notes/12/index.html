2025-05-17

安裝nextcloud

這篇是linux的筆記第12篇

安裝

此安裝是在Portainer下進行安裝

  1. 進入畫面後
  2. 點選Environments底下的環境
  3. 點選左側環境底下的Stacks
  4. 按右上Add stack
  5. 進入畫面在Web editor底下輸入框輸入
volumes:
  nextcloud:
  db:

services:
  db:
    image: mariadb:10.6
    restart: always
    command: --transaction-isolation=READ-COMMITTED --log-bin=binlog --binlog-format=ROW
    volumes:
      - db:/var/lib/mysql
    environment:
      - MYSQL_ROOT_PASSWORD=密碼
      - MYSQL_PASSWORD=密碼
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud

  app:
    image: nextcloud
    restart: always
    ports:
      - 8080:80
    links:
      - db
    volumes:
      - nextcloud:/var/www/html
    environment:
      - MYSQL_PASSWORD=密碼
      - MYSQL_DATABASE=nextcloud
      - MYSQL_USER=nextcloud
      - MYSQL_HOST=db
      - PHP_UPLOAD_LIMIT=100G
      - PHP_MEMORY_LIMIT=8G
  1. 輸入完成後按底下的deploy the stack按鈕
  2. 接著會跑一陣子
  3. 完成後會在containers裡面出現新的容器就表示完成了

設定

在容器的/bin/bash輸入

apt -y update && apt -y install libmagickcore-6.q16-6-extra && apt -y install libsmbclient-dev && pecl install smbclient && echo "extension=smbclient.so" >> /usr/local/etc/php/conf.d/nextcloud.ini

接著重開容器
就能連接外部連結
