linux的筆記-指令

2025-04-15

指令

這篇是linux的筆記第03篇

記錄linux的指令

echo

顯示內容

echo hello

cat

顯示文件內容

cat file

ls

檔案清單

ls

cp

複製檔案

cp file1 file2

mv

搬移檔案

mv file1 file2

touch

建立檔案

touch file

rm

刪除檔案

rm file

cd

變更目前工作目錄

cd ~

mkdir

建立資料夾

mkdir dir

rmdir

刪除資料夾

rmdir dir

cdmod

更改檔案權限

chmod 744 file

grep

找尋字串

grep hello file.txt

less

瀏覽內容

less file.txt

more

跟cat很像顯示於dos介面，但又不像less顯示於另一個畫面，而且可以一頁一頁觀看。

more file.txt

pwd

當前工作目錄位置

pwd

file

檔案資訊

file file.txt

find

找尋檔案

find ~ -name file

sort

排序

sort file
